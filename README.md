# PantryPal 🍲

PantryPal is an AI-powered cooking assistant that helps users find recipes using available ingredients and reduce food waste.

## 🚀 Features
- Generate recipes using AI
- Search recipes based on ingredients
- Simple and user-friendly interface

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Groq API

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Create a `.env` file and add:
   GROQ_API_KEY=your_api_key
4. Run the app:
   python app.py
